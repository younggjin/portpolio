## 개요

앱설명, 사이트 링크 넣기

## 기능

게시판에 구현된 기능 넣기

- 메인 화면 : 판매하고자 하는 상품을 등록.
- 카테고리 : 판매등록한 상품들을 확인하고 관리.

## 라이브러리

사용된 라이브러리 넣기

Thymeleaf Extras Spring Security 6
Lombok
Spring Boot DevTools
Oracle JDBC Driver (ojdbc11)
OAuth2 Client 
MySQL JDBC Driver
